About Me


Hi there! My name is Billy Zhu and I'm currently a Senior Front End Web Developer at Dotdash and am passionate about various Web Technologies. I graduated from the College of Arts and Sciences at Cornell University in 2019 where I majored in Information Science and minored in Computer Science. Projects

Legal Information Institute Web Statistics Kiosk


For my Software Engineering project, I worked with a group of teammates to create a series of web pages depicting usage on Cornell's LII website. I was a key member of the systems subteam which handled the deployment of the web pages onto a Raspberry Pi, configured system dependencies, oversaw security, and improved loading speeds. I also worked on the webscraping subteam which utilized BeautifulSoup in Python to scrape the web for popular news articles pertaining to keywords.

Maddie Freeman Photography


For my Intermediate Web Programming and Development final project, I worked with a group of teammates to create a full-stack web application for a client, Maddie Freeman. We utilized PHP, SQL, HTML, CSS, and Git to help us create our website. Our website provided users with login, uploading, and editing functionalities.

Fully-Pipelined MIPS Processor


For my Systems Engineering project, I worked with my friend to create a fully-pipelined MIPS Processor in Logism. The processor could perform a subset of MIPS instruction such as add, sub, branch, jump, conditionals, etc. We wrote about 1,000 lines of test code in MIPS assembly language to test the circuit.

Handwriting Recognition


For my Artificial Intelligence Practicum project, I worked with a team to create a handwriting recognition system through supervised machine learning. We trained digits and letters through a K-Nearest Neighbors model to predict other digits, letters, and even words!

Temple of Boom


For my Object-Oriented Programming and Data Structures course, I worked on the back-end of a GUI Java game. The game utilizes Depth First Search, Breadth First Search, Heaps, and Dijkstra's Algorithm to help an explorer find gold and make his way back.
